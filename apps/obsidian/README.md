# Obsidian

> ⚠️ Installed via Homebrew

Knowledge base and note-taking app.

## Manual Setup

- [ ] Set up Obsidian (connect vault, configure settings)
- [ ] Run `ob login` to authenticate obsidian-headless (installed via asdf default-npm-packages)
- [ ] Run `ob sync-setup --vault Memex --path ~/Notes/Memex` to link the vault for headless sync

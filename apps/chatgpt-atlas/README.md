# ChatGPT Atlas

OpenAI's web browser with ChatGPT integration.

## Installation

```bash
brew install --cask chatgpt-atlas
```

## Setup

```bash
./apps/chatgpt-atlas/chatgpt-atlas.sh setup
```

This installs ChatGPT Atlas if not already present.

## Manual Setup

Complete these steps after installation:

1. **Sign in** — Launch Atlas and sign in to your OpenAI account
2. **Import data** — Optionally import bookmarks/passwords from another browser
3. **Set as default** — System Settings > Desktop & Dock > Default web browser (if desired)

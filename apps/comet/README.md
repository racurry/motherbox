# Comet

Perplexity's AI-powered web browser.

## Installation

```bash
brew install --cask comet
```

## Setup

```bash
./apps/comet/comet.sh setup
```

This installs Comet if not already present.

## Manual Setup

Complete these steps after installation:

1. **Sign in** — Launch Comet and sign in to your Perplexity account
2. **Import data** — Optionally import bookmarks/passwords from another browser
3. **Set as default** — System Settings > Desktop & Dock > Default web browser (if desired)

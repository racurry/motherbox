# Codex (desktop app)

OpenAI's Codex coding agent packaged as a native macOS app.

## Installation

```bash
brew install --cask codex-app
```

## Setup

```bash
./apps/codex-app/codex-app.sh setup
```

This installs the Codex desktop app if not already present.

## Manual Setup

Complete these steps after installation:

1. **Sign in** — Launch Codex and sign in with your OpenAI account
2. **Grant permissions** — Approve any TCC prompts (Accessibility, Automation, etc.) as needed
